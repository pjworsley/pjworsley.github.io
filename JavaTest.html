Part 3: Building Physical Maps

Phase 1: Finding Genomic Markers

HINT

The first step in building up a physical map is to locate some genetic markers which we already know from previous studies. Genetic markers are sites in the DNA which we already know the location of.

The locations of genetic markers are often found through recombination sequences. Finding the markers in our genomic library and placing them into their known locations is crucial as all the work from here onwards will be based upon our knowledge of these few locations.

BAC Library

HINT

This is the genomic library containing all of the BAC sequences, which in turn means it contains all of the fragmented chromosome.

It is possible to scan this library for genetic markers.

Scan for Genetic Markers

HINT

This is the process of scanning each BAC for previously known genetic markers. Genetic markers are unique sequences found no where else in the genome.

Genetic markers must be known prior to sequencing. Their locations are often found through recombination frequencies or other genetic techniques which allows for their relative placement on a genetic map. Finding one of these markers in a BAC means we will be able to place it in a known location on a map.

Map of Known Genetic Markers on the Genome

HINT

This is a genetic map. It contains the known relative locations of several genetic markers.

This map can be used to help position our BACs if we can find the correct markers within them.

Search Genetic Map of BAC Genetic Markers

HINT

Using the markers found within the BACs it is now possible to search the previously known genetic map for similar markers.

Upon finding a similar marker we can position the respective BAC along side its marker on the map.

Phase 2: BAC End Sequencing

HINT

The next phase of the processes is to begin BAC End Sequencing.

This will allow the build up from the BACs we have already positioned and place further BACs o alongside them.

BAC End Sequence the BAC

HINT

This is the process of BAC end sequencing. It involves sequencing each end of the BAC; this allows us to place the BACs in order by looking for overlaps.

BAC End Sequencing was important during the human genome project as Sanger Sequencing only allowed for a few hundred base pairs to be sequenced at a time. This meant the whole BAC couldn't be reliably sequenced at the same time - they could only sequence the ends.

Phase 3: Shotgun Sequencing

HINT

Shotgun Sequencing is the name given to a technique which involves further fragmentation of the genome. Further fragmentation of each BAC means that the whole BAC can be sequenced and then placed back together.

This technique was used to fill in gaps by the people at the Human Genome Project, however Craig Venter and his team at Celera used Shotgun Sequencing to complete the whole genome.

So far we haven't actually sequenced anything, all we've done is break up a genome! From here on begins the complicated task of putting the genome back together again, and hopefully obtaining a sequence in the process...

All the fragments of DNA that are in the BAC library needed to be pieced back together in the correct order. The result of this process is a physical map of the genome; this map places all of the BACs in the correct order.

This process is carried out in multiple phases:

Before beginning to work on the human genome, researchers did know the sequences and markers of a few specific areas of the genome. Knowing about these areas was essential to be able to begin piecing the human genome together as it provided a starting point.

By looking for genetic markers in BACs it is possible to scan genetic maps for identical genetic markers. If an identical match is found, the BAC can be placed on the genetic map in a known location, thus providing a starting point for sequencing the genome.
