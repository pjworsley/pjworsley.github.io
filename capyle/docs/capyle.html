load_ca(filepath)¶

Load a CA description file

Pre runs the setup function in the description to populate the CAConfig object. Only then will the GUI be properly initialised (states must be known before adding config frame)

Note

If manually specifying a path: sys.path[0] can be used to get the directory of main.py and then ‘/ca_descriptions/xxx.py’ can be appended to create the filepath.

Parameters:filepath (str) – Full path to the CA description py file
