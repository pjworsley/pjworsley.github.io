Part 2: Building Clone Libraries

Human Cells

HINT

Human Cells contain DNA, which can be extracted to create a genomic DNA library. The cells used in the HGP were taken form blood and sperm, which were obtained from a large number of donors around the world.

In reality much of the sequence (>70%) of the reference genome produced came from just a single anonymous male donor from Buffalo, New York. This was to help protect donor identities.

Extract and Purify DNA

HINT

DNA is removed from donor cells, before being further purified and concentrated.

This process is carried out to obtain high quality DNA, free from impurities.

Purified DNA

HINT

DNA which has been purified and concentrated.

At this stage all 23 pairs of chromosomes are still mixed together in the DNA.

Select a Chromosome

HINT

Selection of one, single, chromosome. Only one was picked at a time so sequences obtained from one chromosome would not be matched to sequences from another.

Isolation of chromosomes helped to simplify the enormity of sequences the 3 billion base pairs. Being able to divide these base pairs into groups meant that when it came to piecing together a sequences there were less potential options for each piece to be paired to.

Chromosome

HINT

A single chromosome in isolation.

A human chromosome is a packaged and organized structure made of a considerable amount of human DNA.

Fragment

HINT

Fragmenting DNA is an important step in genome sequencing. Ideally the genome should be fragmented completely randomly and each fragment should be just the right size. Also important is leaving undamaged ends which enable downstream processing.

The main methods used were physical methods (e.g. sonication) and enzymatic methods (e.g. restriction enzymes). While physical methods are much more random than enzymatic methods they can leave damaged ends to fragments. Enzymatic methods only cut at specific locations within DNA, meaning areas of the genome could be missed out, however they leave sticky ends which make the following steps easier to process.

Fragmented Chromosome

HINT

A chromosome which as been broken up into smaller fragments. Each Chromosome was fragmented into smaller pieces, however not all of these pieces are the same size due to the randomness of fragmentation.

Depending on the fragmentation technique used, DNA ends may have to be repaired in order for downstream processing.

Size Selection

HINT

Size selection allows fragments of a desired size to be selected for. One selection method is electrophoresis which separates DNA fragments by mass.

This process is done to ensure no fragment was too big or to small for the DNA vectors which each fragment is place in downstream.

Size Selected Fragment

HINT

These are random fragments of chromosome which are all a similar size.

The HGP used fragments around 150,000 bp pairs in size, or approximately 97,500 kDa.

Empty BAC Vector

HINT

A BAC is a Bacterial Artificial Chromosome, they are circular plasmids constructed with the replication origin of E. coli so they can be maintained in a single copy per cell, as is they were a chromosome. They can hold DNA fragments of up to 300,000 bp.

In early stages of the Human Genome Project, Yeast Artificial Chromosomes (YACs) were also used, however were quickly superseded by BAC vectors, as YAC DNA inserts tended to be unstable due to recombination.

Ligation of Fragment with Vector

HINT

The process of inserting a size selected fragment into empty BACs.

This process was done for every single fragments, ensuring each size selected fragments was contained within its own BAC vector.

BAC Containing Insert

HINT

A BAC containing our size selected fragment.

Once ligation is successful each BAC needs to be placed inside a living host to ensure it will replicate and be maintained.

E.coli

HINT

E.coli is a single celled bacteria in which the BACs being uses are designed to replicate.

E.coli has a doubling time of 20 minutes and will grow in regular lab conditions at 37 degrees.

Transformation into Host E.coli

HINT

The process of getting the BACs into the E.coli host.

Here, BACs can be stored and replicate indefinitely.

Plate and Grow

HINT

Transformed BACs are given time to grown and replicate. This process allows for amplification of our BACs.

Typically, the E.coli containing BACs will be grown on limiting media, on which cells can only survive if they have the correct gene. By only providing these genes on the BAC, and not in the E. coli genome, it is possible to select against E.coli which didn't actual take up a BAC or fragments. This provided a simple screening method.

Select and Store Colonies

HINT

Each colony grown is selected and stored in isolation. It is important to remember that each colony is a clone of an original single E.coli cell and so every colony contains just one single BAC, each of which contains just one single fragment of Human DNA.

All of these colonies are stored for an indefinite length of time. Together they formed a BAC Genomic Library.

A genomic library is a collection of the total genomic DNA from a single organism. The library is in the form of DNA fragments, which are able to be screened, search and used at will. It is literally a library of DNA.

Each colony selected and stored contains hundreds of E.coli cells which all contain the same DNA insert. During the Human Genome Project this process was carried out millions of times to ensure every fragment of DNA could be safely stored. These cells were stored inside BAC Libraries which allowed automated screening of every DNA insert.
